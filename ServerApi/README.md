# Серверный интерфейс (API)

**Что находится в этой папке:**  
Бэкенд-часть проекта (Node.js / Python / Java / Go и т.д.), включая API-эндпоинты, бизнес-логику, модели данных и взаимодействие с БД.

**Планируемые компоненты/модули:**
- RESTful / GraphQL API
- Аутентификация и авторизация (JWT / OAuth2)
- Модуль работы с базой данных (MongoDB / PostgreSQL / MySQL)
- Логирование и обработка ошибок
- Валидация входящих данных
- Документация API (Swagger / Postman)
- WebSocket для real-time функционала (опционально)

**Взаимодействие с другими частями:**
- Принимает запросы от MobileApp и Web
- Отправляет push-уведомления в MobileApp
- Предоставляет единый API для всех клиентов
