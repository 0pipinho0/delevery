# Веб-сайт / Веб-приложение

**Что находится в этой папке:**  
Исходный код веб-версии проекта (React / Vue / Angular / простой HTML+CSS+JS).

**Планируемые компоненты/модули:**
- Публичная часть (лендинг, информация о проекте)
- Личный кабинет пользователя
- Административная панель (если требуется)
- Формы обратной связи
- Интеграция с API сервера

**Взаимодействие с другими частями:**
- Отправляет запросы к ServerApi для получения данных
- Использует те же эндпоинты, что и MobileApp
- Может иметь дополнительный функционал, недоступный в мобильной версии (например, расширенная аналитика)
