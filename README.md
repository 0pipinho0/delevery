# Доставка "Hot delevery"

**Краткое описание идеи: Создать приложение для доставки**  

## Структура репозитория

| Папка         | Назначение                                                                 |
|---------------|---------------------------------------------------------------------------|
| **MobileApp** | Мобильное приложение для пользователей (Android/iOS). Реализует клиентский интерфейс и взаимодействие с сервером через API. |
| **ServerApi** | Серверная часть проекта. Обрабатывает запросы от мобильного приложения и веб-сайта, управляет бизнес-логикой и данными.   |
| **Web**       | Веб-версия приложения или административная панель. Взаимодействует с сервером через те же API, что и мобильное приложение. |
