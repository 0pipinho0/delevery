# Мобильное приложение

**Что находится в этой папке:**  
Исходный код мобильного приложения для платформ Android/iOS (React Native / Flutter / Kotlin / Swift).

**Планируемые компоненты/модули:**
- Экран авторизации и регистрации
- Главный экран с основным контентом
- Профиль пользователя
- Настройки приложения
- Модуль работы с камерой/галереей (если требуется)
- Модуль пуш-уведомлений
- Сетевое взаимодействие с ServerApi

**Взаимодействие с другими частями:**
- Отправляет HTTP-запросы к ServerApi для получения и отправки данных
- Получает пуш-уведомления от ServerApi (Firebase / APNs)
- Веб-версия использует те же API, что гарантирует согласованность данных
